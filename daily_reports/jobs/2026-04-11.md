### 1순위 추천 공고: **당근마켓 - Product Designer 디자인코어**

**왜 최우선인가?**
디자인코어 팀은 "전사 디자인 완성도와 기준을 만드는 팀"으로, 문호균님의 핵심 강점인
**디자인 시스템 구축 경험(서버 22초→0.2초, 코드 40% 감축)** 과 직결됩니다.

#### 칵테일 전략 배합

```
🧪 문제해결 + 시스템 설계    ████████░░  70%
🎨 UI 비주얼 / 심미성         ████░░░░░░  20%
⚡ AI/자동화 / 데이터         ██░░░░░░░░  10%
```

#### JD 미러링 추천 키워드
- **"디자인 완성도"** → 내 프로젝트에서 "품질 기준 수립"으로 미러링
- **"전사에 미치는 영향력"** → 디자인 시스템이 개발팀·PO에 미친 파급효과 강조
- **"기준을 만드는 일"** → 특허 등록(듀얼뷰 동기화 UI)을 "독자적 UX 기준 정립" 표현 활용
- **"사용자와 제품"** → 데이터 기반 의사결정(검색량 +26.4%) 사례 연계

#### 강조할 포트폴리오 프로젝트
1. **디자인 시스템 구축** — 코드 40% 감축, 렌더링 22초→0.2초 (임팩트 수치 전면)
2. **Hubble AI 에이전트** — 1,000개 키워드 → 40초 인사이트 (자동화 + 구조 설계 강조)
3. **검색 UX 개선** — 키워드 +4,752개, 검색량 +26.4% (데이터 검증된 설계)

#### 기업 스테이지별 원하는 디자이너상 (당근)
> 성숙기 플랫폼(월 1,900만 MAU) → **"기준을 세우는 디자이너"**
> 빠른 실험보다 **높은 품질 기준 수립 + 조직 전반에 영향을 주는 시스템 설계자** 선호
