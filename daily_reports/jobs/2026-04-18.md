### 🎯 보이저엑스: Hubble을 무기로 삼아라

**JD 핵심 키워드 분석:**
- `문제 발견` → Hubble 프로젝트: 1,000개 키워드에서 40초 만에 전략 인사이트 도출
- `사용자 중심 UX` → ListeningMind 마이키워드 개선 사례
- `AI 서비스 경험` → Hubble AI 에이전트 설계, 자동화 UX

**칵테일 전략 (보이저엑스 맞춤):**
```
문제해결·AI 설계 ██████████ 50%
디자인 시스템·UX 엔지니어링 ████████ 35%
비주얼·인터랙션 ████ 15%
```

**포트폴리오 프로젝트 우선순위:**
1. `aiAgent_project` (Hubble) — AI 에이전트 UX 설계 경험 → VoyagerX AI 제품과 직결
2. `designSystem_project` — 서버 22초→0.2초, 코드 40% 절감 성과 강조
3. `myKeyword_project` (ListeningMind) — 데이터 기반 의사결정, 검색량 +26.4%

**JD 미러링 용어:**
- "문제를 발견하고 디자인으로 해결"
- "사용자가 더욱 편리하게 사용하도록"
- AI 제품 경험 = Vrew(영상 편집) vs Hubble(키워드 인사이트) → 목적이 다르지만 AI UX 설계 역량 공통점 강조

---

### 🎯 채널톡: B2B SaaS의 언어로 말하라

**JD 핵심 키워드 분석:**
- `HTML/CSS 역량 플러스` → 개발 지식 기반 UX 엔지니어링 스킬 어필
- `B2B SaaS 이해` → 현재 B2B SaaS 스타트업 PO/PM 겸직 경험
- `개발 협업` → React 퍼블리싱 수준의 개발 이해도

**칵테일 전략 (채널톡 맞춤):**
```
문제해결·B2B 이해 ████████ 40%
디자인 시스템·개발협업 ████████ 40%
비주얼·UX 완성도 ████ 20%
```

**포트폴리오 프로젝트 우선순위:**
1. `designSystem_project` — HTML/CSS, 개발협업 강점 완벽 부합
2. `myKeyword_project` — B2B SaaS 제품에서의 UX 개선, 데이터 기반 성과
3. `aiAgent_project` — AI 시대의 채널톡 방향성과 일치, Hubble 자동화 사례
