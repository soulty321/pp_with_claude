Project 2: 당근 모임 개선 (사이드)

01 — Problem Definition

데이터가 가리킨 2회차 방문으로
이어지지 않는 진짜 원인

The Problem

신규 유저가 모임의 핵심 가치를 느끼기 전, '어색함'과 '목표 부재'로 인해 1회성 방문 후 이탈하는 현상 발생.

"기존 멤버들끼리 너무 친해 보여서 끼기 힘들어요."
"뚜렷한 목표가 없으니 한 번 가보고 안 가게 돼요."

[Problem Summary]

  • 월평균 참여자 1.4명 정체
  • 퍼널 분석 결과 1회 참여 후 재방문율 급감
  • 모임 운영 데이터 기반 정량 분석
심리적 장벽을 넘는 '3번의 작은 성공'
02 — User Research

7명의 심층 인터뷰로 발견한
'Aha-Moment'의 조건

Research Insight

Design Thinking 기반 리서치: 5 Whys 심층 인터뷰를 통해 유저가 모임에서 발견하는 핵심 가치를 구조화했습니다.

  • 정성적 발견: 소속감, 루틴 형성, 타인으로부터의 자극(신뢰)이 재방문의 결정적 요인임을 확인.
  • 임계점 도출: 유저는 최소 3회 참여 후에야 '함께'의 본질적 가치를 체감함.

→ 가설: 3회 완주를 돕는 장치가 리텐션의 결정적 열쇠이다.

Empathize → Define → Ideation
03 — Improvement Proposal

3-Step Challenge:
막연한 방문을 '도전'으로 바꾸다

UX Strategy

Hook Model 기반의 3회 출석 게이미피케이션을 통해 신규 유저의 심리적 진입 장벽을 낮추고 완결 욕구를 자극했습니다.

300% Growth
참여자 수 (1.4명 → 4.2명)
